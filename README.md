# 🎓 Student Management System (Java)

## 📌 Project Overview

The **Student Management System** is a Java-based console application designed to manage student records efficiently.

This project allows users to add, view, search, and delete student details using a simple menu-driven interface. It demonstrates the use of **Object-Oriented Programming (OOP)** concepts and Java Collections.

---

## 🚀 Features

### ➕ Add Student
- Add new student records
- Store student ID, name, age, and department

### 👀 View Students
- Display all registered students
- Shows complete student information

### 🔍 Search Student
- Search students using Student ID
- Displays matching student details

### 🗑️ Delete Student
- Remove student records using Student ID

### 🚪 Exit
- Safely exit the application
